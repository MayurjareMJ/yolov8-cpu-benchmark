# Phase 4: Benchmark PyTorch

```python
import time
import cv2
from ultralytics import YOLO

model=YOLO("yolov8n.pt")

img=cv2.imread("test.jpg")

for _ in range(20):
    model(img)

start=time.time()

for _ in range(100):
    model(img)

end=time.time()

fps=100/(end-start)

print("FPS:",fps)
print("Latency:",1000/fps)
```

Measure:

* FPS
* Latency

# Expected Findings on Your CPU

For an Intel i5-1335U, typical relative results are:

| Format        | Relative Speed |
| ------------- | -------------- |
| PyTorch       | 1x             |
| ONNX          | 1.2x - 1.5x    |
| OpenVINO      | 1.5x - 2.5x    |
| OpenVINO INT8 | 2x - 4x        |

OpenVINO and INT8 are where you'll likely see the biggest gains because Intel CPUs are heavily optimized for them.
